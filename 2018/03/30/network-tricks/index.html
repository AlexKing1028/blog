network tricks

Posted on 2018-03-30 | In machine learning |
神经网络的一些优化方法

network compression

  1. 低秩矩阵，近似原有的w矩阵，例如使用svd分解、Toeplitz矩阵配合Krylov分解近似原矩阵
    • 参考：Exploiting linear structure within convolutional networks for efﬁcient evaluation.
  2. 剪枝，在训练结束后将不重要的神经元或滤波器去除
  3. 量化+权值共享。
    • 通过kmeans对同一层的权值分类，每一类权值只存储其kmeans中心点，整个权值矩阵只有k个不同的值，用$logk$比特存储。
    • 同一类权值共享一个值，在update时也保持一致
    • 参考：Deep compression: Compressing DNNs with pruning, trained quantization and huffman coding.
  4. 结构设计。
    • bottleneck和1*1卷积（减少通道数，实际上就是将各个通道按不同的权值累加）
    • 分组卷积
    • 扩展卷积，dilated convolution，跳着卷积，扩大感受野
  5. 用小模型模仿大模型
    • 小模型去模仿大模型的中间层和结果的输出
    • knowledge distillation，A Gift From Knowledge Distillation: Fast Optimization, Network Minimization and Transfer Learning，让小模型和大模型的层间关系（两层通道的内积）相同。

imbalanced learning

类不平衡会导致

  1. 模型对少样本类训练较差。精度假象，少样本类别召回率低
  2. 不好设置概率阈值

采样

通过采样调整不同类别的样本数量

  1. 过采样：对小样本类别随机抽取多次，直到和大样本数量相当。对小样本重复采样容易引入噪声。
  2. 欠采样：从大样本随机抽取与小样本数量相当的数据。样本个数不够。
  3. SMOTE采样：通过插值对小样本进行补充。随机选一个样本作为中心点a，计算k近邻，从中随机选一个点b，用这两个点插值出一个新的点。

阈值

调整阈值和样本类个数相对应，但是这需要假设样本分布和总体分布一致。

集成学习

  1. bagging：bootstrap aggregation，在原样本集上随机可重选择k个样本，组成新的训练集，这样重复n次，分别训练n个模型，再聚合预测。
  2. boosting：每次迭代分类器将重点放在被错误分类的那一类上。
