关于instance segmentation任务的研究工作的survey

proposal based

Simultaneous Detection and Segmentation（2014）

Based on R-CNN

  1. MCG, get region proposal and foreground(replace for selective search)
  2. feather extraction, two branch of CNN
  3. classification, linear svm
  4. refinement, logistic regression+cnn

DeepMask

DeepMask对每一个图片的patch x（或者说proposal）经过VGG提取特征之后，分为两个分支

  1. segmentation branch：获得512个feature map，在经过两次全连接得到一张mask，经过upsampling（具体是bilinear interpolation）获得原patch大小的mask。
  2. score branch：多层全连接后得到一个score，表示该patch是否为正例。

what is a positive sample?

正例必须满足以下两个条件：

  1. instance的中心也在patch的中心。
  2. patch必须完全包含instance

train

上式为loss function，总共k个training patch，第一项为mask的loss，只训练正例部分，第二项为score loss。

训练样本由每张图片ground truth的多个instance的bounding box组成。实际训练中，为了保证模型的稳定性，patch和instance的中心和大小允许一定程度的偏移。

test/inference

密集的选取不同scale和location的patch

instanceFCN

score map in sliding window

sliding window为图中绿色大框。作者将sliding window分割为$k \times k$等分，每个区域对应instance的身体部位，分别对全图生成一个score map，仅检测图中存在的所有部位。因此对每个proposal，对应输出有$k \times k$个channel，分别对应每个区域的检测结果。

例如图中有两个人，区域4的feature map对身体的左边识别比较高。

instance assembling

在得到$k \times k$个特征图之后，我们将第i个特征图的区域i放入新的map中，组成$k \times k$大小的新的map，作为对该proposal的mask。

model structure

网络先通过13层的vgg提取特征，分为两个支路，上一个支路负责预测mask，下一个支路负责预测bounding box。在获得$k \times k$个feature channel后，将每个选取的sliding window分割为$k \times k$块，分别填上对应channel的feature。

train

随机选取n个sliding window，预测mask和判断是否为bounding box的正例。损失函数如下：

$$loss = \sum_i(L(p_i,p_i*)+\sum_{i,j}L(S_{i,j},S_{i,j}*))$$

p表示sliding window是否为正例，S表示每个pixel的label，其中带*表示ground truth，L为logistic regression loss。

实际训练对样本进行缩放，随机选择256个sliding window进行训练。

test

为了保证精度，按固定stride密集的选取sliding window，根据score和mask使用NMS筛选得到最终结果。

FCIS

全名fully convolutional instance sementic segmentation

主要是基于instance FCN的改进版，主要有三点：

  1. 由于两个任务有密切的关联，detection和segmentation统一到一个分支。之前大部分工作是两个任务共享提取feature的map，然后分成两个分支进行训练
  2. 增加类别，instance FCN只能识别背景和前景（即物体），FCIS能够识别c个类别的物体和背景。
  3. 使用RPN获得ROI，相比instance FCN增加了尺寸和形状

procedure

  1. CNN提取feature
  2. RPN获得N个ROIs
  3. 对于每个ROI，获得$2k^2 \times (C+1)$个channel的score map，系数2是包括每个pixel的detection和segmentation的score，k表示ROI分割的块数，C为类别数，加1表示背景。
    • detection score表示该pixel是否在该类别的instance的bounding box中。
    • segmentation score表示该pixel是否在该类别的前景。
  4. 经过instance assembling获得$2(C+1)$个channel
  5. 对这$C+1$个类别，每一个预测detection score和segmentation score
    • detection score：对2个score map先逐pixel取max，再计算全图的平均值作为ROI为该类别的score，对$C+1$个类计算一个softmax
    • segmentation：对2个score map逐pixel计算softmax，segmentation大于detection的为mask

PANet

bottom-up augmentation

FPN证明了高层级的特征结合低层级可以获得到更精确的位置信息，但是不同层级之间隔了太多层，不利于获得位置信息。

augmented path

在FPN的基础上增加了一层自底向上的downsample的层，即part b网络部分。和FPN相似，下一层卷积和平行层经过lateral connection的结果逐个元素累加。不同层之间总共只隔了不超过10层。

Adaptive feature pooling

FPN用于Fast RCNN时，对不同大小的ROI，分别使用不同的特征层做预测。作者认为各个特征层都对ROI的预测存在影响。因此作者提出任意ROI都在所有层计算特征区域，再利用ROIAlign+max/sum pooling获得最终结果。
